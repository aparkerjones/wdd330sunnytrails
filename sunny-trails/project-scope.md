# Project Scope (MVP)

## Problem
Planning National Park trips requires switching across multiple websites for park details, weather, and alerts.

## Solution
Sunny Trails Planner combines those planning details into one place so users can compare options and save trip plans quickly.

## In-Scope (MVP)

- Search National Parks by name and state
- View park details (description, location, images, visitor info)
- View weather forecast for selected park
- View park alerts (closures, warnings)
- Create, edit, and delete itineraries
- Add notes and a gear checklist
- Persist itineraries and notes in local storage
- Show trip summary before save

## Out of Scope (For Now)

- User authentication and accounts
- Cloud sync
- Social features
- Payments/booking flow

## Success Criteria

- User can search and choose a park in under 60 seconds
- User can save itinerary and retrieve it after refresh
- Weather + alerts load reliably with clear fallback errors
- Core flows work on mobile and desktop
